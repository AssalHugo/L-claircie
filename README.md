# L-claircie
L'Éclaircie est le baromètre de la redevabilité politique. Face à la complexité législative, notre application agit comme la météo de l'Assemblée nationale : elle compare automatiquement les promesses de campagne des groupes politiques avec leurs votes réels dans l'hémicycle. Un grand soleil pour une promesse tenue, un orage pour une contradiction.
